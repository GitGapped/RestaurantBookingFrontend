# Restaurant booking

A full-featured React Native mobile application for managing restaurants, reservations, and books. Built with Expo and React Navigation.



## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI
- iOS Simulator (for Mac) or Android Studio (for Android development)

## Installation

1. Clone the repository

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Run on your preferred platform:
```bash
# For iOS
npm run ios
# For Android
npm run android
# For web
npm run web
```
